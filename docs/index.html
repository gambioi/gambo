FAQ

Is it safe?

Gambo is open source — the full code is on GitHub. It runs locally in your own client and never asks for your password or token.

How do I update?

Open the Gambo Updater inside Discord → "Check for Updates" → "Update Now" → restart. New plugins and fixes arrive automatically.

How do I uninstall?

Run the installer again and click Uninstall. It cleanly restores your normal Discord.

Will it work after Discord updates?

Yes. Gambo re-injects itself and the crew ships fixes fast whenever Discord changes something.
